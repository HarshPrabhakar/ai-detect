About

About The Project

Problem Statement

With the rapid advancement of Generative AI tools like MidJourney, DALL-E, and Stable Diffusion, AI-generated images have become indistinguishable from real photographs to the human eye. This poses serious risks in misinformation, deepfake fraud, and digital identity theft.

This project proposes a deep learning-based solution that can accurately classify images as real or AI-generated, providing a reliable tool for media verification.
